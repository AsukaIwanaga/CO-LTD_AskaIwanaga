# メール報告 共通フォーマットルール

iPhone 12 の Gmail アプリ（縦向き）で快適に読めることを最優先とする。

---

## デバイス仕様

- iPhone 12: 画面幅 390pt
- Gmail アプリ: 左右パディング約20pt → 本文領域 約350pt
- 等幅フォント想定: 1文字 約9-10pt
- **有効文字幅: 最大33文字（半角）**
- 全角文字は2文字分: 全角のみなら約16文字

---

## 基本ルール

### 1. 横幅

- **1行の最大幅: 半角33文字**（全角16文字相当）
- テーブル・数値の並びもこの幅に収める
- はみ出す場合は改行して次行にインデントで続ける

### 2. 区切り

- 大セクション: `=================================`（33文字）
- 中セクション: `---------------------------------`（33文字）
- 小セクション: `-----  ラベル  -----`

### 3. フォント

- 全てプレーンテキスト（text/plain）で送信
- HTML メールは使わない
- Gmail のプロポーショナルフォント表示を前提
- 幅制御はセパレータ（33文字）のみ。本文の折り返しは Gmail に任せる

### 4. 数値の配置

- ラベル左寄せ、数値右寄せ
- 金額はカンマ区切り + 通貨記号
- パーセントは符号付き（±##.#%）

```
ラベル     右寄せ数値
AMT        ###,###
T/C            ###
LBR%        ##.##%
```

### 5. リスト

- 箇条書きは `・` または `[ ]` を使用
- インデントは半角2スペース

```
・項目1
  詳細テキスト
・項目2
```

### 6. 日付・時刻

- 日付: `MM-dd (ddd)` または `MM/dd`
- 時刻: `HH:mm`

### 7. タイトル行

- 全大文字英字 + 日付
- 例: `MORNING BRIEFING  03-31 (Tue)`

### 8. セクション内の空行

- セクション間: 2行空け
- サブセクション間: 1行空け
- セクション内データ: 空行なし

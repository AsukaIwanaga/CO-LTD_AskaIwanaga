# 夜のブリーフィング（Evening Report）

Don の1日を締めくくるための記録・確認セッションです。

## フロー

### STEP 1: 表示

```
╔══════════════════════════════════════════╗
║   EVENING REPORT — YYYY/MM/DD (曜日)     ║
╚══════════════════════════════════════════╝
```

### STEP 2: 記録確認（順番に聞く）

1. **今日のログ**
   「今日のログを記録しますか？」
   → 「はい」なら `/general:log` へルーティング

2. **習慣チェック**
   「今日達成できた習慣を教えてください。（スキップは「なし」）」
   → `habit_log.json` に追記

3. **残高・財務記録**
   「今日の残高を記録しますか？（メイン・ローン・クレジット）」
   → `/finance:financial` へルーティング

4. **支出記録**
   「今日の支出はありますか？（なければスキップ）」
   → `/finance:budget` へルーティング

5. **ランニング記録**（走った日のみ）
   「今日ランニングしましたか？」
   → 「はい」なら `/hr:running` へルーティング

6. **タスク更新**
   「完了したタスクや新しいタスクはありますか？」
   → `/sales:task` へルーティング

7. **メモ・気づき**
   「今日気になったこと・記録しておきたいことはありますか？」
   → あれば `/general:memo` へルーティング

### STEP 3: 締め

全記録完了後：
> 「Don、お疲れ様でした。ゆっくり休んでください。」

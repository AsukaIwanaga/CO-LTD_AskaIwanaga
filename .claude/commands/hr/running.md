# ランニング記録（開発部）

ランニングの記録追加・統計確認を管理します。
データ: `/Users/askaiwanaga/LIFE/robco-terminal/data/running_log.json`

## データ構造
```json
[{
  "id": "1",
  "date": "2026-03-18",
  "distance": 5.2,
  "duration": 30,
  "memo": "気持ちよかった"
}]
```

## 操作メニュー

### 1. ランニングを記録する
ユーザーに以下を確認してから追記する：
- 距離（km、例: 5.2）
- 時間（分、例: 30）
- メモ（任意）
- 日付（デフォルト: 今日）

id は既存の最大値 +1（データが空の場合は "1"）。

記録後にペースを計算して表示する：
```
🏃 記録完了！
  日付: YYYY-MM-DD
  距離: X.X km
  時間: XX 分
  ペース: X'XX" /km
```

### 2. 記録一覧・統計を確認する
全記録を表示し、統計を算出する：

```
🏃 ランニング記録
─────────────────────────
日付        距離   時間   ペース
2026-03-18  5.2km  30分   5'46"/km
─────────────────────────
累計距離:  XX.X km
累計回数:  X 回
平均ペース: X'XX"/km
今月の距離: XX.X km
```

## 注意事項

- ペースの計算: duration（分）/ distance（km）→ 分:秒 表示
- 距離は小数点1桁で保存・表示する

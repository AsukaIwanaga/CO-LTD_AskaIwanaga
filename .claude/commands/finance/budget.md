# 予算・支出管理（経理部）

月次予算と支出を管理します。
データ: `/Users/askaiwanaga/LIFE/robco-terminal/data/budget.json`

## データ構造
```json
{
  "monthly": { "2026-03": { "食費": 30000, "交通費": 10000 } },
  "expenses": [
    { "id": "1", "date": "2026-03-18", "category": "食費", "amount": 1200, "description": "ランチ" }
  ]
}
```

## 操作メニュー

### 1. 今月のサマリーを確認
`budget.json` を読み込み、今月（YYYY-MM）の予算と支出を比較して表示する：

```
💰 {YYYY年MM月} 予算サマリー
─────────────────────────
カテゴリ    予算      使用額    残り
食費       30,000   12,500   17,500  ██████░░░░ 42%
交通費     10,000    3,200    6,800  ███░░░░░░░ 32%
─────────────────────────
合計       40,000   15,700   24,300
```

### 2. 支出を記録する
ユーザーに以下を確認してから `expenses` 配列に追記する：
- カテゴリ（既存カテゴリを提示する）
- 金額（数値）
- 説明（任意）
- 日付（デフォルト: 今日）

id は既存の最大値 +1。

### 3. 今月の支出一覧を表示
今月の支出を日付降順で表示する：
```
📋 {MM月}の支出一覧
─────────────────────────
MM/DD  [カテゴリ]  ¥X,XXX  説明
```

### 4. 予算を設定する
カテゴリと金額をユーザーに確認し、`monthly.YYYY-MM` に設定する。

## 注意事項

- 更新前に既存データを必ず読み込む
- 金額は整数で保存する

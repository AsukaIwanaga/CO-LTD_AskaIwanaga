# 株価確認（経理部）

Yahoo Finance API でウォッチリストの株価をリアルタイム取得します。

## デフォルトウォッチリスト

| ティッカー | 銘柄 |
|---|---|
| ^N225 | 日経平均 |
| ^DJI | ダウ工業株 |
| ^GSPC | S&P 500 |
| 7203.T | トヨタ自動車 |
| 6758.T | ソニーグループ |
| BTC-USD | ビットコイン |

## フロー

1. 各ティッカーについて Yahoo Finance v8 API で価格を取得する：
   ```
   https://query1.finance.yahoo.com/v8/finance/chart/{TICKER}?interval=1d&range=1d
   ```
2. レスポンスから現在値・前日比・変化率を抽出する
3. 以下のフォーマットで表示する：

```
📈 株価ウォッチリスト — HH:MM 更新
─────────────────────────────────
日経平均    ¥38,500.00   +250.00  (+0.65%)  ↑
ダウ        $41,200.00  -100.00  (-0.24%)  ↓
S&P500      $5,600.00    +30.00  (+0.54%)  ↑
トヨタ      ¥3,200.00    +15.00  (+0.47%)  ↑
ソニー      ¥2,800.00    -20.00  (-0.71%)  ↓
BTC         $85,000.00 +1,000.00 (+1.19%)  ↑
```

## 注意事項

- APIが失敗したティッカーは「取得失敗」と表示してスキップする
- 株式市場が閉まっている場合は前日終値を表示する
- 金額の通貨は銘柄に応じて ¥ または $ を使う

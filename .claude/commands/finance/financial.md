# 資産管理（経理部）

残高・ローン・クレジット残高のスナップショットを管理します。
データ: `/Users/askaiwanaga/LIFE/robco-terminal/data/financial.json`

## データ構造
```json
[{
  "id": "1",
  "date": "2026-03-18",
  "timestamp": "2026-03-18T21:00:00",
  "main": 107835,
  "loan": 1000000,
  "credit": 0,
  "description": ""
}]
```

## 操作メニュー

### 1. 最新の資産状況を確認
`financial.json` を読み込み、最新エントリを表示する：

```
🏦 資産状況（最終更新: YYYY-MM-DD）
─────────────────────────
普通預金:    ¥XXX,XXX
ローン残高: -¥X,XXX,XXX
クレジット: -¥XXX,XXX
─────────────────────────
純資産:      ¥XXX,XXX
前回比:      +¥X,XXX
```

### 2. 新しいスナップショットを記録する
ユーザーに以下を確認してから追記する：
- 普通預金残高（main）
- ローン残高（loan）
- クレジット利用残高（credit）
- メモ（description、任意）

id は既存の最大値 +1。date と timestamp は現在日時。

### 3. 推移を確認する
全エントリを日付順に表示し、純資産（main - loan - credit）の推移を示す：
```
📈 純資産推移
─────────────────────────
2026-03-14  ¥107,835  →  純資産: -¥892,165
2026-03-15  ¥100,000  →  純資産: -¥900,000
```

## 注意事項

- 純資産 = main - loan - credit
- 更新前に既存データを必ず読み込む

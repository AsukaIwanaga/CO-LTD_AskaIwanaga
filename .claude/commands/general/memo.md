# メモ管理（開発部）

メモの作成・閲覧・削除を管理します。
データ: `/Users/askaiwanaga/LIFE/robco-terminal/data/memos.json`

## データ構造
```json
[{
  "id": "1",
  "text": "メモ本文",
  "category": "未分類",
  "timestamp": "2026-03-18T21:00:00"
}]
```

## 操作メニュー

ユーザーに以下を確認してください：

### 1. メモ一覧を確認
`memos.json` を読み込み、カテゴリ別に表示する：
```
📝 メモ一覧
─────────────────────────
【仕事】
  1. [テキスト冒頭30文字...] (2026-03-18)
【未分類】
  2. [テキスト冒頭30文字...] (2026-03-17)
```

### 2. メモを作成する
ユーザーに本文・カテゴリを確認してから追記する：
- 本文をユーザーから受け取る
- カテゴリ（デフォルト: 未分類）
- id は既存の最大値 +1
- timestamp は現在時刻

### 3. メモを詳細表示する
番号またはIDを指定してもらい、全文を表示する。

### 4. メモを削除する
対象メモをユーザーに確認してから削除する。

## 注意事項

- 削除前に対象メモの内容をユーザーに確認する
- カテゴリは既存のものを提示し、新規カテゴリも受け付ける

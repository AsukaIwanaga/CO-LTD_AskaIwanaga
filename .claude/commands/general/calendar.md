# カレンダー（営業部）

Google Calendar の確認・予定追加を行います。MCP の gcal ツールを使用します。

## 操作メニュー

ユーザーに以下を確認してください：

1. **今日の予定を確認** → `gcal_list_events` で本日の予定を取得して表示する
2. **今週の予定を確認** → `gcal_list_events` で今週の予定を取得して表示する
3. **予定を追加する** → タイトル・日時・場所をユーザーに確認してから `gcal_create_event` で作成する
4. **空き時間を確認する** → `gcal_find_my_free_time` で空き時間を探す
5. **予定を更新・削除する** → `gcal_get_event` で対象を特定してから操作する

## 表示フォーマット

```
📅 {日付}の予定
─────────────────────
HH:MM — [タイトル] (場所)
HH:MM — [タイトル]
予定なし
```

## 注意事項

- 予定の作成・削除は実行前にユーザーに内容を確認する
- タイムゾーンは Asia/Tokyo を使用する

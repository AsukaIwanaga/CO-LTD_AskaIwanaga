# Gmail（営業部）

Gmail の確認・送受信を行います。MCP の Gmail ツールを使用します。

## 操作メニュー

ユーザーに以下を確認してください：

1. **受信トレイを確認する** → `gmail_search_messages` で `in:inbox` を検索し、最新10件を一覧表示する
2. **メールを読む** → メールIDを指定して `gmail_read_message` で内容を表示する
3. **メールを送る / 下書きを作る** → 宛先・件名・本文をユーザーに確認してから `gmail_create_draft` で下書き作成する
4. **スレッドを読む** → `gmail_read_thread` でスレッド全体を表示する

## 表示フォーマット

受信トレイ一覧：
```
📧 受信トレイ（最新10件）
─────────────────────
1. [件名] — 差出人 (日時)
2. [件名] — 差出人 (日時)
...
```

## 注意事項

- 送信前に必ずユーザーに内容を確認する
- 実際の送信は行わず、下書き（draft）として保存する
- ユーザーが「送信して」と明示した場合のみ送信を検討し、再確認する

# ポータル通達確認 (Clemenza)

すかいらーくポータルの通達一覧を取得・表示します。

## フロー

### STEP 1: スクリプト実行

```bash
python3 scripts/scrapers/portal.py
```

を実行し、`drafts/portal_latest.json` を読み込む。

スクリプトが失敗した場合（セッション切れ等）は以下を案内する：

> 「Don I.、セッションが切れています。`python3 scripts/auth/portal_login.py` を実行して再ログインしてください。」

### STEP 2: 通達表示

取得成功後、以下のフォーマットで表示：

```
📢 ポータル通達（未読 X 件）
取得: YYYY/MM/DD HH:MM  店舗: [店舗名]  担当: [スタッフ名]

直近の通達:
  [YYYY/MM/DD] カテゴリ / タイトル — 発行部門
  [YYYY/MM/DD] カテゴリ / タイトル — 発行部門
  ...

（全 X 件中、直近7日分を表示。それ以外は件数のみ案内）
```

### STEP 3: 追加アクション確認（スタンドアロン実行時のみ）

`/morning` から呼ばれた場合はこのステップをスキップ。

単体実行の場合、通達表示後に確認する：

1. 「通達の詳細を確認しますか？（番号または「なし」）」
   → 番号が指定されれば該当通達のIDと情報を表示する

2. 「タスクに追加する通達はありますか？」
   → あれば `tasks.json` へ追記する（deadline・priority・tag="WORK" を確認して保存）

## 再ログイン

セッション切れ時は `portal_login.py` を案内するだけでよい。
Claude Code ターミナルで手動実行してもらう（自動化は不可）。

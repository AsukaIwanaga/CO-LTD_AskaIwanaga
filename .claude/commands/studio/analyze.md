# /studio:analyze — Moe Greene（分析チーム / analyzer）

YouTube の再生数・CTR・視聴維持率を取得・分析し、次回企画へのフィードバックを出す。

---

## フロー

### STEP 1: データ取得

```bash
python3 studio/scripts/analyze_youtube.py
```

YouTube Data API v3 で以下を取得：
- 視聴回数・いいね数・コメント数
- CTR（クリック率）
- 視聴維持率（平均視聴割合・ドロップポイント）
- トラフィックソース（検索 / 関連動画 / 外部等）

### STEP 2: レポート生成

```
📊 分析レポート — YYYY-MM-DD_タイトル
─────────────────────────────────────
視聴回数:      X,XXX 回
CTR:           X.X%（業界平均: 2〜5%）
平均視聴率:    XX%（XX分XX秒 / 全XX分）
ドロップ率高:  X:XX 付近（PART XX の冒頭）

改善提案:
・オープニングを XX 秒以内に短縮する
・PART XX の説明が長い → 削減推奨
・タイトルに「2026年版」等の時事性を追加する
```

### STEP 3: Jack へフィードバック

分析結果を台本生成の参考情報として渡す：
> 「Moe Greene から Jack へ: 次回企画では以下を考慮してください。[改善点]」

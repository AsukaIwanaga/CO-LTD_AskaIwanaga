# WILL: 定期タスクの自動検知・再生成

**起票**: 2026-03-31
**起票者**: 経営企画部
**reported**: 2026-03-31

## 課題
Google Tasks API に繰り返し機能がなく、ブラウザで設定した繰り返しのみ。
Don がブラウザで設定し忘れた定期タスクは再生成されない。

## 提案
task_sync.py で「過去に同名タスクが繰り返し完了されているパターン」を検知し、
次の期限のタスクを自動生成する仕組みを追加。
例: 「WRK_mngt_商品管理」が毎週完了 → 翌週分を自動作成。

## 期待効果
Google ブラウザでの繰り返し設定に依存しなくなる。
